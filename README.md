# 🌐 Personal Portfolio Website

A modern and responsive **personal portfolio website** built using **HTML5, CSS3, and JavaScript**. This portfolio showcases my technical skills, projects, certifications, education, and contact information with a clean UI, smooth animations, and an interactive user experience.

---

## 🚀 Live Demo

🔗 **Website:** https://my-portfolio-opal-one-hczqxlr0kf.vercel.app

---

## 📌 Features

* 🎨 Modern glassmorphism-inspired UI
* 🌙 Dark and Light theme toggle
* 📱 Fully responsive design for desktop, tablet, and mobile
* ⚡ Smooth scrolling navigation
* ✨ Scroll-based animations and interactive effects
* 👨‍💻 About Me section
* 🛠️ Skills showcase
* 📂 Projects section with GitHub and Live Demo links
* 🎓 Education and Certifications
* 📧 Functional contact form powered by Web3Forms
* 📄 Resume download option

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* Web3Forms (Contact Form)
* Google Fonts

## 🚀 Getting Started

### Clone the repository

```bash

```

### Navigate to the project folder

```bash

```

### Open the project

Simply open **index.html** in your preferred web browser.

No additional dependencies or installation are required.
